# README

Question application:
create questions for you friend.


* Installation:

1) Clone respected git repository

2) Install all dependencies
bundle i

3) Create db and migrate schema
rake db:create
rake db:migrate

4) Make fake db data
rails db:seeds

5) Now run your application
rails s


* Versions:

Ruby  3.0.0

Rails 7.0.1

Bootstrap 5.1.3
